# Newstalk
I created this project with the intention of listening to some article while I do other stuff. Like coding, playing some games and whatnot

## Features
Yet to be finished

## Requirements
Python version3.14, libraries needed trafilatura, gtts, playsound3


## Usage
Just paste the article/newspaper link on the code and run it.

## How it works
The program will fetch the link, read the text, ignore images, ads and whatnot, only focused on the text. Then It will copy the text and get the AI to read it aloud.
